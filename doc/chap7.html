7.1-1 RepresentationCategoryObject
‣ RepresentationCategoryObject( c, C, str )( operation )

Returns: an object in G-mod

There are 3 arguments. The first argument is The first argument is an irreducible character c, which will be then interpreted as giving the input [ [ 1, c ] ]. The second argument is the CAP category C = G-mod. The output is the unique simple object in G-mod having corresponding to the character c. The third argument is a string used as follows: SetString( GIrreducibleObject( c ), str ).
